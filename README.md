## Overview
MSME Intelligence Index is a frontend analytical dashboard designed to
visualize and analyze micro, small, and medium enterprise ecosystem
indicators.
The platform focuses on:
- MSME ecosystem trends
- business intelligence visualization
- policy-oriented analytics
- economic indicator representation
- structured data presentation
## Features
- Responsive dashboard UI
- Interactive data visualization
- Multi-section analytical layout
- Modern TailwindCSS interface
- Mobile-friendly responsive design
- Production deployment using Vercel
## Tech Stack
- React
- Vite
- TailwindCSS
- GitHub
- Vercel
